SmartDigest ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how we handle information when you use our Chrome extension for AI-powered content summarization.

## üîí Data Collection & Storage

### What We Collect

**We do not collect, store, or transmit any personal information about you.**

### What We Store Locally

The following information is stored locally on your device using Chrome's storage API:

- **Gemini API Key**: Your Google Gemini API key for AI summarization
- **TTS Settings**: Your text-to-speech preferences (voice selection, speed)
- **Extension Settings**: Basic configuration preferences

### Data Processing

- **Text Summarization**: When you request a summary, the selected text is sent to Google's Gemini API for processing
- **No Storage**: We do not store, log, or retain any of the text you summarize
- **Temporary Processing**: Text is only processed for the duration of the API request

## ü§ñ Google Gemini API Integration

### API Usage

- **Service Provider**: Google Gemini AI (Google AI Studio)
- **Purpose**: Text summarization and content analysis
- **Data Sent**: Only the text you select for summarization
- **Data Retention**: Google's Gemini API may retain data according to their privacy policy

### Google's Privacy Policy

When using SmartDigest, you are also subject to Google's Privacy Policy for Gemini AI services. Please review [Google's Privacy Policy](https://policies.google.com/privacy) for information about how Google handles your data.

### What's Stored

- API key (encrypted by Chrome)
- Voice preferences
- Speed settings
- Basic extension configuration
